## 📦 O que é este repositório?

Este vault é um **guia de estudos completo de japonês**, construído inteiramente dentro do [Obsidian](https://obsidian.md/) — um aplicativo de notas em Markdown que funciona com arquivos locais, sem depender de nuvem.

Todo o conteúdo está disponível em **português e inglês**, tornando o aprendizado acessível independentemente do seu idioma de preferência.

---

## 🚀 Como começar

### 1. Clone o repositório com o GitHub Desktop

1. Baixe e instale o **[GitHub Desktop](https://desktop.github.com/)**
2. Clique em **File → Clone Repository**
3. Cole a URL deste repositório e escolha onde salvar no seu computador
4. Clique em **Clone**

### 2. Abra como vault no Obsidian

1. Baixe e instale o **[Obsidian](https://obsidian.md/)**
2. Na tela inicial, clique em **"Open folder as vault"**
3. Selecione a pasta onde você clonou o repositório
4. Pronto — todas as notas, links e plugins estarão prontos para uso ✅

---

## 🧠 Como o aprendizado funciona aqui

Este vault usa o plugin **Spaced Repetition** (repetição espaçada) integrado ao Obsidian. Isso significa que o sistema vai te mostrar os flashcards nos **momentos ideais antes de você esquecer**, maximizando a retenção com o mínimo de tempo de estudo.

> O método de repetição espaçada é cientificamente comprovado e usado por políglotas do mundo todo. Aqui ele está embutido diretamente nas suas anotações.

## 📚 Conteúdo do vault

### 🔤 Hiragana
O primeiro passo no japonês. O hiragana é o silabário base da língua, usado para palavras nativas japonesas e gramática. As notas cobrem:

- Todos os 46 caracteres base
- Caracteres combinados (dakuten, handakuten)
- Pronúncia com exemplos em português e inglês
- Flashcards para memorização com repetição espaçada

---

### 🔡 Katakana
O segundo silabário, usado principalmente para palavras de origem estrangeira e empréstimos linguísticos. As notas cobrem:

- Todos os 46 caracteres base
- Paralelo visual com o Hiragana para facilitar a memorização
- Exemplos de palavras do cotidiano
- Flashcards com repetição espaçada

---

### 📖 As 800 Palavras Mais Comuns do Japonês
Baseado no vídeo **[800 Japanese Words for Everyday Life](https://www.youtube.com/watch?v=tkMOsYrtV74)** do canal **JapanesePod101**, referência mundial no ensino de japonês para iniciantes.

Cada palavra inclui:

| Campo | Descrição |
|---|---|
| 🇯🇵 Japonês | Escrita em kana e/ou kanji |
| 🔊 Romaji | Pronúncia romanizada |
| 🇧🇷 Português | Tradução em português |
| 🇺🇸 Inglês | Tradução em inglês |
| 🃏 Flashcard | Disponível para revisão com spaced repetition |

---

### 🃏 Flashcards
Todas as palavras, caracteres e frases possuem flashcards no formato compatível com o plugin **Spaced Repetition** do Obsidian. Basta abrir o painel de revisão e estudar — o algoritmo cuida do resto.

---

### 💬 Frases do Cotidiano
Além do vocabulário isolado, o vault inclui frases reais usadas no dia a dia, com:

- Escrita em japonês (hiragana / katakana / kanji)
- Romaji para pronúncia
- Tradução em **português**
- Tradução em **inglês**
- Contexto de uso (cumprimentos, compras, transporte, restaurante, etc.)

## 📌 Dicas para estudar bem

1. **Estude todo dia**, mesmo que por 10 minutos — a consistência vence a intensidade
2. **Revise os flashcards primeiro** ao abrir o Obsidian
3. **Não pule o hiragana** — é a base de tudo
4. **Ouça o vídeo do JapanesePod101** enquanto acompanha as notas
5. **Escreva à mão** os caracteres — ajuda muito na memorização
